# awuawu

An Education game designed to teach CS and AI literacy. Inspired by https://trex-runner.com/

# how to run

make sure you are in node v18.15.0

add the OpenAI API key to the index.js file

apiKey: <YOUR_KEY>,

```
npm ci
npm start
```
